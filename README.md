# string-utils

[![npm version](https://img.shields.io/npm/v/string-utils.svg)](https://www.npmjs.com/package/string-utils)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Common string utilities — truncate, capitalize, pad, template

## Installation

```bash
npm install string-utils
```

## Functions

- `truncate` — Truncate string with ellipsis
- `capitalize` — Capitalize first letter of each word
- `padLeft` — Pad string on the left
- `padRight` — Pad string on the right
- `template` — Simple string template with {{key}} placeholders
